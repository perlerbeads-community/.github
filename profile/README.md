# Perler Beads Community

**The open community hub for [perlerbeads.io](https://perlerbeads.io)** — a platform built for perler bead artists, pattern designers, and creative enthusiasts around the world.

---

## What Is This?

We're building tools and a community that make perler bead art more accessible, expressive, and fun. This GitHub organization is the home for:

- Feedback and feature requests for [perlerbeads.io](https://perlerbeads.io)
- Open discussions about tools, techniques, and creative ideas
- Community-maintained resources: color charts, guides, and tool links
- Collaboration between artists and builders

## Community Values

- **Open** — ideas from everyone are welcome, regardless of skill level
- **Focused** — we stay on topic: perler beads, pixel art, and the tools around them
- **Constructive** — criticism is great; cruelty is not
- **Patient** — this is a small team with a lot of love for the craft
