# Contributing to perlerbeads Community

Welcome! We're glad you're here. There are many ways to contribute — you don't need to write code.

---

## Ways to Contribute

### Non-Technical
- **Vote on features** — browse [open issues](https://github.com/perlerbeads-community/feedback/issues) and add 👍 to things you want
- **Report bugs** — something broken? [Tell us →](https://github.com/perlerbeads-community/feedback/issues/new?template=bug_report.yml)
- **Share ideas** — [feature requests](https://github.com/perlerbeads-community/feedback/issues/new?template=feature_request.yml) or [discussions](https://github.com/orgs/perlerbeads-community/discussions)
- **Add resources** — color charts, tool links, guides in the [resources repo](https://github.com/perlerbeads-community/resources)

### Technical
- Look for issues labeled [`help-wanted`](https://github.com/perlerbeads-community/feedback/labels/help-wanted) or [`good-first-issue`](https://github.com/perlerbeads-community/feedback/labels/good-first-issue)
- Fork → branch → PR
- All PRs require one maintainer review before merge

---

## Community Standards

- Be kind and constructive
- Stay on topic: perler beads, pixel art, and related tools
- Search before posting to avoid duplicates
- One idea per issue — it keeps discussions focused

Full Code of Conduct: [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md)
